ArtCOA is a free tool by Smalta Art Gallery that issues a two-page Certificate of Authenticity for an artwork or a limited-edition print, with a QR code for online verification. It runs entirely in your browser — no account needed.

What the program does

  • Page 1 — Certificate of Authenticity: all artwork data, identifiers, QR code and signature lines.
  • Page 2 — Artwork identification: the artwork image plus key identifiers and signature lines.
  • A QR code on page 1 links to a verification page that carries the certificate data inside the link.

How to create a certificate

  1. Pick the language (English / Українська / Slovenčina) at the top left.
  2. Fill in artist, title, year, status, technique, support, sheet size, artwork area.
  3. The Edition ID builds automatically: initials-year-artworkNo-code-sizeCm-editionSize (e.g. sd-2026-001-EDH-50x70-15). Initials are suggested from the artist's name and can be edited.
  4. The Certificate no. = Edition ID + copy number (e.g. …-15-1). Both are filled in automatically.
  5. The Edition code is set automatically from the Status (see table below).
  6. Optionally set gallery and date.
  7. On page 2, attach the artwork image ("Choose file").
  8. Paste your private signing key (generate it once in keys.html). Keep it secret — only this key can issue valid certificates.
  9. Check the live preview on the right (two A4 pages).

Save as PDF

Click Save as PDF → the print dialog opens → choose destination “Save as PDF”. The text is crisp and selectable (vector). For a clean result, in More settings set Margins: None and turn off Headers and footers (one-time). The file is named after the certificate number.

Send & share

Send PDF shares a ready file: on phones it opens the native share sheet (email, messengers, AirDrop); on desktop it downloads the file and opens an email draft. Copy verify link copies the verification URL to the clipboard.

QR verification

Scanning the QR opens the verification page, which checks the signature automatically against the gallery’s public key — the buyer needs no secret. A valid, untampered certificate shows “Authentic”.

Security. Certificates are signed with an ECDSA P-256 key pair: the gallery signs with its private key (kept secret); anyone verifies automatically with the public key built into the verify page. Forgery is computationally infeasible without the private key, and any change to the data invalidates the signature (tamper-evident). Limitations: the data still travels inside the QR link, and there is no central registry / revocation yet — that comes with the server version.

Languages

The interface and form appear in the selected language. On the certificate itself, labels are English for English, and bilingual (“selected / English”) for Ukrainian and Slovak.

Privacy & data

With your consent we record each generated certificate (its details, device info and approximate location/IP) to improve the service. You can decline and still use everything; change your choice anytime via the “Privacy & data” link in the generator. We don’t sell this data.

ArtCOA — безкоштовний інструмент від Smalta Art Gallery, що створює двосторінковий сертифікат автентичності для твору мистецтва чи тиражного відбитка з QR-кодом для онлайн-перевірки. Працює повністю у браузері — без реєстрації.

Що робить програма

  • Сторінка 1 — сертифікат автентичності: усі дані твору, ідентифікатори, QR-код і лінії для підписів.
  • Сторінка 2 — ідентифікація твору: зображення твору, ключові ідентифікатори та підписи.
  • QR-код на 1-й сторінці веде на сторінку перевірки; дані сертифіката «зашиті» в посилання.

Як створити сертифікат

  1. Оберіть мову (English / Українська / Slovenčina) вгорі зліва.
  2. Заповніть художника, назву, рік, статус, техніку, основу, розмір аркуша, розмір зображення.
  3. Edition ID формується автоматично: ініціали-рік-№твору-код-розмірСм-тираж (напр. sd-2026-001-EDH-50x70-15). Ініціали підставляються з імені художника й редагуються.
  4. № сертифіката = Edition ID + номер копії (напр. …-15-1). Обидва заповнюються автоматично.
  5. Код видання підставляється автоматично за статусом (див. таблицю).
  6. За потреби вкажіть галерею та дату.
  7. На 2-й сторінці додайте зображення твору («Вибрати файл»).
  8. Вставте приватний ключ підпису (згенеруйте одноразово в keys.html). Тримайте в таємниці — лише цей ключ може видавати дійсні сертифікати.
  9. Перевірте живий перегляд праворуч (дві сторінки A4).

Збереження у PDF

Натисніть Зберегти як PDF → відкриється діалог друку → оберіть призначення «Зберегти як PDF». Текст чіткий і виділяється (вектор). Для чистого результату у Додаткових налаштуваннях поставте Поля: Немає і вимкніть Колонтитули (одноразово). Файл називається за номером сертифіката.

Надсилання та поширення

Надіслати PDF ділиться готовим файлом: на телефоні відкриває рідне меню «Поділитися» (пошта, месенджери, AirDrop); на комп'ютері завантажує файл і відкриває чернетку листа. Копіювати лінк копіює посилання для перевірки.

Перевірка через QR

Сканування QR відкриває сторінку перевірки, яка автоматично звіряє підпис із публічним ключем галереї — покупцю не потрібен секрет. Дійсний і незмінений сертифікат показує «Автентичний».

Безпека. Сертифікати підписуються парою ключів ECDSA P-256: галерея підписує приватним ключем (тримається в таємниці); будь-хто перевіряє автоматично публічним ключем, вбудованим у сторінку перевірки. Без приватного ключа підробка криптографічно неможлива, а будь-яка зміна даних робить підпис недійсним (виявлення підробки). Обмеження: дані досі в QR-посиланні, і поки немає центрального реєстру / відкликання — це буде в серверній версії.

Мови

Інтерфейс і форма показуються обраною мовою. На самому сертифікаті підписи англійською — для англійської, і двомовні («обрана / English») — для української та словацької.

Приватність і дані

За вашою згодою ми фіксуємо кожен створений сертифікат (його дані, інформацію про пристрій та приблизне місцезнаходження/IP) для покращення сервісу. Можна відхилити й користуватися далі; змінити рішення — посиланням «Приватність і дані» в генераторі. Ми не продаємо ці дані.

ArtCOA je bezplatný nástroj od Smalta Art Gallery, ktorý vytvára dvojstranový certifikát pravosti pre umelecké dielo alebo limitovanú tlač s QR kódom na online overenie. Beží úplne vo vašom prehliadači — bez registrácie.

Čo program robí

  • Strana 1 — certifikát pravosti: všetky údaje diela, identifikátory, QR kód a riadky na podpis.
  • Strana 2 — identifikácia diela: obraz diela, kľúčové identifikátory a podpisy.
  • QR kód na 1. strane vedie na overovaciu stránku; údaje certifikátu sú vložené v odkaze.

Ako vytvoriť certifikát

  1. Vyberte jazyk (English / Українська / Slovenčina) vľavo hore.
  2. Vyplňte umelca, názov, rok, stav, techniku, podklad, rozmer listu, plochu diela.
  3. Edition ID sa zostaví automaticky: iniciály-rok-čDiela-kód-rozmerCm-náklad (napr. sd-2026-001-EDH-50x70-15). Iniciály sa navrhnú z mena umelca a dajú sa upraviť.
  4. Č. certifikátu = Edition ID + číslo kópie (napr. …-15-1). Obe sa vyplnia automaticky.
  5. Kód edície sa nastaví automaticky podľa stavu (pozri tabuľku).
  6. Voliteľne zadajte galériu a dátum.
  7. Na 2. strane priložte obraz diela („Vybrať súbor“).
  8. Vložte svoj súkromný podpisový kľúč (vygenerujte ho raz v keys.html). Uchovajte ho v tajnosti — iba tento kľúč môže vydávať platné certifikáty.
  9. Skontrolujte živý náhľad vpravo (dve strany A4).

Uloženie do PDF

Kliknite na Uložiť ako PDF → otvorí sa dialóg tlače → vyberte cieľ „Uložiť ako PDF“. Text je ostrý a označiteľný (vektor). Pre čistý výsledok v Ďalších nastaveniach nastavte Okraje: Žiadne a vypnite Hlavičky a päty (jednorazovo). Súbor sa pomenuje podľa čísla certifikátu.

Odoslanie a zdieľanie

Odoslať PDF zdieľa hotový súbor: na telefóne otvorí natívne menu zdieľania (e-mail, čety, AirDrop); na počítači stiahne súbor a otvorí koncept e-mailu. Kopírovať odkaz skopíruje overovací odkaz.

Overenie cez QR

Naskenovanie QR otvorí overovaciu stránku, ktorá automaticky overí podpis pomocou verejného kľúča galérie — kupujúci nepotrebuje žiadne tajomstvo. Platný a nezmenený certifikát zobrazí „Pravé“.

Bezpečnosť. Certifikáty sú podpísané párom kľúčov ECDSA P-256: galéria podpisuje súkromným kľúčom (drží sa v tajnosti); ktokoľvek overí automaticky verejným kľúčom zabudovaným v overovacej stránke. Bez súkromného kľúča je falšovanie výpočtovo neuskutočniteľné a akákoľvek zmena údajov podpis zneplatní (odhalenie manipulácie). Obmedzenia: údaje sú stále v QR odkaze a zatiaľ nie je centrálny register / zrušenie — to príde so serverovou verziou.

Jazyky

Rozhranie a formulár sa zobrazujú vo zvolenom jazyku. Na samotnom certifikáte sú popisy v angličtine pre angličtinu a dvojjazyčné („zvolený / English“) pre ukrajinčinu a slovenčinu.

Súkromie a údaje

S vaším súhlasom zaznamenávame každý vytvorený certifikát (jeho údaje, informácie o zariadení a približnú polohu/IP) na zlepšenie služby. Môžete odmietnuť a naďalej všetko používať; voľbu zmeníte odkazom „Súkromie a údaje“ v generátore. Tieto údaje nepredávame.
